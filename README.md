# 🏫 6학년 우리반 홈페이지

우리 반 친구들을 위한 즐거운 온라인 공간입니다! 👋

## 📝 프로젝트 소개
이 홈페이지는 6학년 우리 반 친구들이 학교 소식을 확인하고, 가정통신문을 쉽게 볼 수 있도록 만들었습니다.

## 🚀 주요 기능
- **홈 화면**: 반 친구들을 환영하는 메시지
- **가정통신문**: 준비물, 현장체험학습 등 중요 알림 확인
- **로그인/회원가입**: (데모) 사용자 인증 화면 체험

## 🛠️ 사용 기술
- HTML5
- CSS3 (Google Fonts 'Jua' 사용)
- JavaScript (Vanilla JS)

## 💻 실행 방법
1. 이 저장소(Repository)를 다운로드합니다.
2. `Index.html` 파일을 웹 브라우저(크롬, 엣지 등)로 엽니다.
